HOW TO USE
==========
Aplikasi ini berfungsi untuk mencatat tugas atau pekerjaan yang harus anda lakukan nanti atau besok hari. Aplikasi ini juga dapat mengingatkan anda akan tugas tersebut melalui fitur notifikasi.

Aplikasi ini belum selesai sepenuhnya karena tugas yang bisa diingatkan hanya dalam jangka waktu hari ini dan besok. Namun aplikasi ini tetap dapat digunakan sebagai pencatat tugas seperti pada umumnya.

Jika tugas sudah anda kerjakan, maka tekan kotak yang ada disamping kiri tugas untuk mencoret tugas / menyelesaikan tugas. Tombol dapat ditekan kembali jika tugas belum selesai.

Untuk menambah tugas, klik tombol "+" pada pojok kanan bawah layar dan isikan nama tugas, jam pengingat, hari (besok atau sekarang), serta perlu diingatkan atau tidak. Jika sudah, maka tekan tombol "done" untuk menyimpannya.

Untuk menghapus tugas, klik tombol "tempat sampah" pada bagian kanan masing masing tugas dan tugas akan secara otomatis dihapus. Ingat bahwa tuags yang dihapus tidak dapat dikembalikan lagi.

Jika ingin menghilangkan tuags yang sudah selesai, maka hanya perlu menekan tombol "Hide Completed" di bagian pojok kanan atas, dibawah tombol informasi. Tekan lagi untuk mengembalikannya.

NOTE : Tugas yang dibuat untuk besok hari tidak akan dapat ditandai selesai sebelum besok hari.
NOTE : Tugas yang belum selesai hari ini akan tetap berada pada bagian TODAY untuk mengingatkan ada.

Tombol informasi akan membuka halaman tentang aplikasi dan identitas pengembang.
